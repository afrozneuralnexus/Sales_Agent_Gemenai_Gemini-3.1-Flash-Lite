# Gemini Flash-Lite RAG Chatbot

A simple Retrieval-Augmented Generation (RAG) chatbot using:

- Gemini API
- LangChain
- FAISS
- Streamlit

## Features

- PDF Upload
- Vector Search
- Question Answering
- Streamlit Frontend

## Installation

```bash
pip install -r requirements.txt
```

## Run

```bash
streamlit run app.py
```

## Add API Key

Create `.env`

```env
GOOGLE_API_KEY=your_api_key
```
